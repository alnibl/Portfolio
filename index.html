DataMind: Сервис обработки бухгалтерских документов

Пилотная серверная система для анализа PDF-документов с использованием OCR и LLM.

  • Развёртывание Ubuntu + Docker + FastAPI
  • Интеграция EasyOCR, Tesseract, LLM (Qwen3, DeepSeek)
  • API загрузки и анализа документов
  • Точность классификации: 97.5%, Извлечение номера/даты: ~90%
  • Обработка до 10 документов одновременно за 30–50 сек

Технологии: Python 3.12, Docker, FastAPI, EasyOCR, Tesseract, Ollama, Qwen, DeepSeek, Gunicorn, Nginx, Poetry, OpenCV

Пример API:

curl -X POST http://x.x.x.x/api/upload \
  -F "files=@/path/Акт.pdf" \
  -F "files=@/path/Договор.pdf"

GET http://x.x.x.x/api/result/{pdf_id}
GET http://x.x.x.x/download/{pdf_id}.json
